# NestJS 项目总结

## 项目概述

`api-photo-serve` 是一个基于 NestJS 框架开发的图片服务系统，提供了完整的图片上传、管理、分享等功能。项目采用前后端分离架构，后端使用 NestJS + TypeScript，前端使用 Vue 3 + TypeScript。

## 技术栈

### 1. 后端技术栈

- **核心框架**：NestJS
- **编程语言**：TypeScript
- **数据库**：MySQL
- **ORM**：TypeORM
- **认证**：JWT + Passport
- **文件上传**：Multer
- **实时通信**：WebSocket
- **邮件服务**：Nodemailer
- **任务队列**：Bull
- **API 文档**：Swagger
- **测试框架**：Jest
- **容器化**：Docker

### 2. 前端技术栈

- **核心框架**：Vue 3
- **编程语言**：TypeScript
- **构建工具**：Vite
- **状态管理**：Pinia
- **UI 框架**：Element Plus
- **CSS 框架**：TailwindCSS
- **HTTP 客户端**：Axios
- **路由管理**：Vue Router
- **国际化**：vue-i18n
- **图表库**：ECharts
